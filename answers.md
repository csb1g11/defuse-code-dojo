### Part 1
```
1.  ['white', 'red', 'green', 'white']  = Defused
2.  ['white', 'orange', 'green', 'white'] = Not Defused
```

### Part 2
```
1.  ['white', 'white', 'red', 'white', 'orange', 'black', 'black', 'green', 'orange']  = Defused
2.  ['white', 'white', 'green', 'orange', 'green'] = Invalid move, Not Defused
3.  ['white', 'white', 'red', 'red', 'red', 'white', 'white', 'black', 'green', 'orange'] = Defused
4.  ['white', 'black', 'black', 'black', 'black', 'green', 'orange'] = Invalid move, Not Defused
5.  ['black', 'green', 'green'] = Invalid move, Not Defused
6.  ['red', 'red', 'white', 'orange', 'black', 'green'] = Not Defused
```

### Part 3
```
 1.  {'white': 4, 'red': 3, 'black': 4, 'green': 1, 'orange': 1} = Can be defused
 2.  {'white': 4, 'red': 3, 'black': 4, 'green': 0, 'orange': 1} = Cannot be defused
 3.  {'white': 3, 'red': 1, 'black': 48, 'green': 1, 'orange': 2} = Can be defused
 4.  {'white': 3, 'red': 1, 'black': 48, 'green': 1, 'orange': 1} = Cannot be defused
```
